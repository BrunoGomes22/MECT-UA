# Part II readme

## How to Run: 

1. To run testAudio:
```bash
    g++ -g testAudio.cpp -o testAudio -lsfml-audio -lsfml-system

    ./testAudio -f <audio_file_name1> <audio_file_name2> ... -i|-w|-h|-q <bits>
    -i: Display basic info
    -w: Display waveform
    -h: Display histogram
    -q: Quantization
   ```
